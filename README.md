# 🌐 Portfolio Website - Rajdeep Majumdar

🚀 **Live Demo:** https://rajdeep7.vercel.app

---

## 📌 Overview

A modern and responsive personal portfolio website built to showcase skills, projects, and experience as a **MERN Stack Developer**. The website emphasizes clean design, smooth animations, and an engaging user experience.

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🎬 Framer Motion
* 🌐 Vercel (Deployment)

---

## ✨ Features

* 💻 Fully responsive design (mobile + desktop)
* 🎯 Smooth animations using Framer Motion
* 🖱️ Custom cursor for enhanced interactivity
* ⚡ Fast performance and optimized UI
* 📂 Project showcase with live links
* 📄 Clean and modern UI/UX
* 🔗 Seamless navigation

---

## 📸 Sections Included

* 🏠 Home
* 👨‍💻 About Me
* 🛠️ Skills
* 📂 Projects
* 📞 Contact
